\subsection{User Profile and Settings}
Beside all the main features, we also provide a user profile page
for users to view their profile and update settings.

\subsubsection{Features}
Below are some features for user profile
\begin{itemize}%[leftmargin=*]
    \item Users should be able to view their user information
    \item Users should have an interface to view their team information
    \item Drop down menu to change the settings
\end{itemize}
