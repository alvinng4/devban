\subsection{Calendar}
Calendar is an extension of Kanban, providing a visualization for Kanban tasks with deadlines.

\subsubsection{Features}
Below are some features for Calendar:
\begin{itemize}%[leftmargin=*]
    \item User should be able to quickly see dates with deadline tasks
    \item User should be able to see a list of deadline tasks on a selected date
    \item User should be able to navigate between different dates easily
    \item User should be able to create a task with selected date as deadline
\end{itemize}
