\section{Introduction}

\textbf{Devban} is a collaborative task management application designed specifically for development teams. This initial release (v1.0.0) combines Kanban-style task organization, calendar scheduling, AI-powered assistance, and gamification elements to help teams track and complete their work efficiently.

\section{What's New in Version 1.0.0}

This is the initial release of Devban, featuring a comprehensive set of tools for team collaboration and task management.

\subsection{Core Features}

\subsubsection{Authentication and User Management}
\begin{itemize}
    \item \textbf{Email/Password Authentication}: Secure sign-up and sign-in with email and password
    \item \textbf{Google Sign-In}: Quick authentication using Google accounts
    \item \textbf{Password Recovery}: Forgot password functionality with email reset
    \item \textbf{Account Management}: User profile management and account deletion
\end{itemize}

\subsubsection{Team Collaboration}
\begin{itemize}
    \item \textbf{Team Creation}: Create new teams with custom names and optional license IDs
    \item \textbf{Team Joining}: Join existing teams using invite codes
    \item \textbf{Invite Code System}: Generate time-limited invite codes (7-day validity) for team members
    \item \textbf{Team Roles}: Admin and Member roles with appropriate permissions
    \item \textbf{Team Information}: View team details, member count, and role information
\end{itemize}

\subsubsection{Kanban Board}
\begin{itemize}
    \item \textbf{Three-Column Layout}: Organize tasks in To-do, In Progress, and Done columns
    \item \textbf{Task Creation}: Create tasks with comprehensive details:
    \begin{itemize}
        \item Title and description (Markdown supported)
        \item Difficulty levels (Very Easy, Easy, Normal, Hard, Very Hard)
        \item Progress tracking (0-100\%)
        \item Deadline assignment
        \item Task pinning
    \end{itemize}
    \item \textbf{Task Management}: View, edit, and delete tasks
    \item \textbf{Drag and Drop}: Move tasks between columns (on wide screens)
    \item \textbf{Responsive Layout}: Adapts to different screen sizes (iPhone, iPad)
\end{itemize}

\subsubsection{Calendar Integration}
\begin{itemize}
    \item \textbf{Monthly Calendar View}: Visual representation of task deadlines
    \item \textbf{Date Selection}: Tap dates to view associated tasks
    \item \textbf{Task Visualization}: Visual indicators for dates with deadlines
    \item \textbf{Quick Task Creation}: Create tasks directly from the calendar view
    \item \textbf{Navigation}: Easy month-to-month navigation with Today button
\end{itemize}

\subsubsection{AI Assistant (AskLLM)}
\begin{itemize}
    \item \textbf{LLM-Powered Chat}: Interactive AI assistant for development help
    \item \textbf{Streaming Responses}: Real-time response generation
    \item \textbf{Code Syntax Highlighting}: Properly formatted code blocks with syntax highlighting
    \item \textbf{Markdown Support}: Rich text formatting in responses
    \item \textbf{Conversation Management}: Clear context, restart conversation, and stop generation
    \item \textbf{Session History}: Maintain conversation context during active sessions
\end{itemize}

\subsubsection{Team Discussion}
\begin{itemize}
    \item \textbf{Real-Time Messaging}: Communicate with team members instantly
    \item \textbf{Markdown Support}: Format messages with Markdown
    \item \textbf{Message History}: View chronological message history
    \item \textbf{User Attribution}: See who sent each message with timestamps
\end{itemize}

\subsubsection{Gamification System}
\begin{itemize}
    \item \textbf{Experience Points (EXP)}: Earn EXP by completing tasks
    \item \textbf{Difficulty-Based Rewards}: 
    \begin{itemize}
        \item Very Easy: 5 EXP
        \item Easy: 10 EXP
        \item Normal: 20 EXP
        \item Hard: 40 EXP
        \item Very Hard: 80 EXP
    \end{itemize}
    \item \textbf{Level System}: Level up based on accumulated EXP (Level = (EXP / 100) + 1)
    \item \textbf{Progress Tracking}: Visual progress bar showing EXP and level progress
    \item \textbf{User Profile}: Display current level and experience in profile
\end{itemize}

\subsubsection{User Interface and Experience}
\begin{itemize}
    \item \textbf{Tab-Based Navigation}: Four main tabs (Home, Calendar, AskLLM, Profile)
    \item \textbf{Adaptive Layout}: Responsive design for different screen sizes
    \item \textbf{Theme Customization}: Multiple color themes (Blue, Green, Orange, etc.)
    \item \textbf{Dark Mode Support}: Auto, Dark, or Light mode options
    \item \textbf{Sound Effects}: Optional sound feedback for actions
    \item \textbf{Haptic Feedback}: Tactile feedback for interactions
    \item \textbf{Neobrutalism UI}: Modern, bold design aesthetic
\end{itemize}

\section{Getting Started}

\subsection{Initial Setup}
\begin{enumerate}
    \item Download and install the application
    \item Launch the app and create an account or sign in with Google
    \item Create a new team or join an existing team using an invite code
    \item Start creating tasks and collaborating with your team
\end{enumerate}

\subsection{Quick Start Guide}
\begin{itemize}
    \item \textbf{Create Your First Task}: Navigate to Home tab, tap the ``+'' button in any column
    \item \textbf{Set a Deadline}: Use the Calendar tab to visualize and manage task deadlines
    \item \textbf{Ask for Help}: Use the AskLLM tab to get AI assistance with development questions
    \item \textbf{Collaborate}: Use the Discussion section in the Home tab to communicate with your team
    \item \textbf{Customize}: Visit the Profile tab to adjust settings, theme, and preferences
\end{itemize}

\section{Known Limitations}

\begin{itemize}
    \item \textbf{Team Membership}: Users can only belong to one team at a time
    \item \textbf{Offline Support}: Limited offline functionality; internet connection required for most operations
    \item \textbf{Platform Support}: Currently available only for iOS devices
    \item \textbf{Invite Codes}: Invite codes expire after 7 days and can only be used once
    \item \textbf{Task Assignments}: Tasks are team-scoped but don't have individual assignees in this version
\end{itemize}
